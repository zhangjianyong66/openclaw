## 完整部署方案

### 一、下载 frp

**两台机器都执行：**

```bash
wget https://github.com/fatedier/frp/releases/download/v0.52.3/frp_0.52.3_linux_amd64.tar.gz
tar -xzf frp_0.52.3_linux_amd64.tar.gz
cd frp_0.52.3_linux_amd64
sudo cp frps /usr/local/bin/   # 服务端用
sudo cp frpc /usr/local/bin/   # 客户端用
```

---

### 二、服务端配置（公网服务器）

**1. 创建配置目录和文件：**

```bash
sudo mkdir -p /etc/frp
sudo nano /etc/frp/frps.ini
```

**2. 写入配置：**

```ini
[common]
bind_port = 7000
vhost_http_port = 8080
vhost_https_port = 8443
dashboard_port = 7500
dashboard_user = admin
dashboard_pwd = your_password_here
```

**3. 配置 systemd 服务：**

```bash
sudo nano /etc/systemd/system/frps.service
```

写入：

```ini
[Unit]
Description=FRP Server
After=network.target

[Service]
ExecStart=/usr/local/bin/frps -c /etc/frp/frps.ini
Restart=always
RestartSec=5
User=root

[Install]
WantedBy=multi-user.target
```

**4. 启动服务：**

```bash
sudo systemctl daemon-reload
sudo systemctl enable frps
sudo systemctl start frps
sudo systemctl status frps
```

---

### 三、客户端配置（内网机器）

**1. 创建配置目录和文件：**

```bash
sudo mkdir -p /etc/frp
sudo nano /etc/frp/frpc.ini
```

**2. 写入配置（根据你的实际情况修改）：**

```ini
[common]
server_addr = openclaw.zhangjianyong.top
server_port = 7000

[openclaw]
type = http
local_port = 18789
custom_domains = openclaw.zhangjianyong.top
```

**3. 配置 systemd 服务：**

```bash
sudo nano /etc/systemd/system/frpc.service
```

写入：

```ini
[Unit]
Description=FRP Client
After=network.target

[Service]
ExecStart=/usr/local/bin/frpc -c /etc/frp/frpc.ini
Restart=always
RestartSec=5
User=root

[Install]
WantedBy=multi-user.target
```

**4. 启动服务：**

```bash

sudo systemctl enable frpc
sudo systemctl start frpc
sudo systemctl status frpc
```

### 五、Nginx 转发到 frps（支持 HTTP + WebSocket）

在公网服务器上，配置 nginx，把 80 端口转发到 frps 的 `vhost_http_port`（这里是 8080）：

```nginx
# 80 端口：强制跳转到 HTTPS
server {
    listen       80;
    listen       [::]:80;
    server_name  openclaw.zhangjianyong.top;

    return 301 https://$host$request_uri;
}

# 443 端口：终止 TLS，并反代到 frps 的 vhost_http_port（这里是 8080）
server {
    listen       443 ssl;
    listen       [::]:443 ssl;
    server_name  openclaw.zhangjianyong.top;

    ssl_certificate     /usr/local/nginx/conf/cert/openclaw.zhangjianyong.top.pem;
    ssl_certificate_key /usr/local/nginx/conf/cert/openclaw.zhangjianyong.top.key;
    ssl_session_timeout 5m;
    ssl_ciphers         ECDHE-RSA-AES128-GCM-SHA256:ECDHE:ECDH:AES:HIGH:!NULL:!aNULL:!MD5:!ADH:!RC4;
    ssl_protocols       TLSv1.1 TLSv1.2 TLSv1.3;
    ssl_prefer_server_ciphers on;

    location / {
        proxy_pass http://127.0.0.1:8080;

        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;

        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
```
